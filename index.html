Reality Brief:
Automated News

After becoming fed up with overly opinionated and biased news sources, I created Reality Brief to provide a simple list of daily high-impact events. Reality Brief is an attempt to curb the inherent bias in most news sources with indiscriminate, automated reporting. By making use of various APIs and web scrapers, it collects lists of specific types of events, and reports every time one of those events occurs. It is a purely client-side application, making use of Javascript, jQuery and JSONP. You can view Reality Brief in action here, and view my code on Github here.

Socrates is Mortal:
A sketchbook for logicians

Socrates is Mortal is a platform for logical argument development. Users can create Syllogisms, and the app will determine the validity of those Syllogisms. Once created, arguments can be shared with others to demonstrate the reasoning behind a belief.

I was inspired to start this project because of the countless arguments on the internet I see every day that could been resolved if only one person had clearly laid out the reasoning behind their beliefs. This app is still in Beta but I am very excited about its potential.
